# ADR-010. Fair Usage

Date: 2026-01-29

## Context
We have to codify what it means to use our service fairly. We can't have people using it for illegal activities -- that much is obvious -- but what about for creating chatbots or integrating into coding tools like cursor?

### Other fair usage policies
All major providers -- OpenAI, Anthropic, Google -- have fair usage policies. I think we can essentially copy and paste one of those for the illicit activities section.

We note that there should be possible exceptions for certain research directions
